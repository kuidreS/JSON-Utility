== Overview

JSON Utility application provides REST API endpoints for main JSON operations, such as:

* Validate JSON
* Merge two JSONs
* Sort JSON
* Search in JSON
* Generate Type Definition
* JSON Pretty Print
* Validate JSON Schema
* Convert JSON to CSV
* Convert CSV to JSON
* Convert JSON to EXCEL
* Convert EXCEL to JSON
* Convert JSON to XML
* Convert XML to JSON
* Convert JSON to YML
* Convert YML to JSON
